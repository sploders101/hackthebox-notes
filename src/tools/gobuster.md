# Gobuster

Gobuster is a web enumeration tool that can brute-force

* file paths
* virtual hosts (`Host: ...` header)
* dns
* public S3 buckets

## Example usage

Use gobuster to enumerate common paths on the server:

```
sploders101@htb[/htb]$ gobuster dir -u http://10.10.10.121/ -w /usr/share/seclists/Discovery/Web-Content/common.txt
```

```
sploders101@htb[/htb]$ gobuster dns -d inlanefreight.com -w /usr/share/SecLists/Discovery/DNS/namelist.txt
```
