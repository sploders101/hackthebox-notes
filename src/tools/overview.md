# Overview

## Discovery & Scanning

| Tool           | Description                                                             |
|----------------|-------------------------------------------------------------------------|
| `nmap`         | Network scanning utility with simple scripts for getting more details   |
| `gobuster`     | Web enumeration tool for brute forcing of paths, virtual hosts, and dns |
| `whatweb`      | Finds versions of web servers, frameworks, and applications             |
| seclists       | Repository of common wordlists for various purposes                     |
| `searchsploit` | Searches databases for known exploits                                   |

## Remote Filesystem Manipulation

| Tool        | Description                                                     |
|-------------|-----------------------------------------------------------------|
| `ftp`       | Manipulates files over the File Transfer Protocol (ftp)         |
| `sftp`      | Manipulates files over the Secure File Transfer Protocol (SFTP) |
| `smbclient` | Manipulates files over the Server Message Block (SMB) protocol  |

## Documentation

| Tool         | Description                                 |
|--------------|---------------------------------------------|
| `eyewitness` | Web screenshotter & default password finder |

## Specialty Protocols

| Tool | Description                                                   |
